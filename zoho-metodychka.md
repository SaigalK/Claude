## Що таке Zoho

Zoho — це екосистема з 55+ бізнес-додатків в одному акаунті. Альтернатива Microsoft 365 + Salesforce + QuickBooks разом, але дешевше і в одному вікні.

**Головна ідея:** всі відділи працюють в одній системі — продажі, фінанси, проекти, HR, пошта — і дані між ними синхронізуються автоматично.

## Крок 1 — Реєстрація (15 хвилин)

1. Зайди на **[zoho.com](https://www.zoho.com)**
2. Натисни **Sign Up for Free**
3. Вкажи корпоративний email (`@sanekoбуд` або Gmail)
4. Обери **Zoho One** (пробний 30 днів — всі додатки)

> Якщо хочеш тільки CRM — обирай **Zoho CRM Free** (до 3 користувачів безкоштовно назавжди)

## Крок 2 — Zoho CRM (найважливіше)

### Що це дає
- Всі клієнти і контакти в одному місці
- Воронка угод: Лід → Контакт → Угода → Виграно/Програно
- Нагадування про дзвінки і зустрічі
- Історія всіх комунікацій з клієнтом

### Перше налаштування (30 хвилин)

**1. Додай перших клієнтів**
```
CRM → Контакти → Новий контакт
Заповни: Ім'я, Компанія, Телефон, Email
```

**2. Створи воронку продажів під будівництво**
```
Налаштування → CRM → Pipelines → Новий Pipeline
Стадії: Запит → Огляд об'єкту → КП відправлено → Договір → В роботі → Завершено
```

**3. Додай першу угоду**
```
CRM → Угоди → Нова угода
Вкажи: Назва проекту, Клієнт, Сума, Очікувана дата закриття
```

### Ключові поля для будівництва
- **Тип об'єкту:** житловий / комерційний / промисловий
- **Адреса об'єкту**
- **Кошторисна вартість**
- **Відповідальний менеджер**

## Крок 3 — Zoho Books (фінанси)

### Що це дає
- Виставлення рахунків і актів
- Контроль оплат (хто заплатив, хто ні)
- Витрати і собівартість
- Звіти: P&L, Cash Flow, Баланс

### Перше налаштування (45 хвилин)

**1. Налаштуй компанію**
```
Books → Налаштування → Профіль організації
Заповни: Назва, ЄДРПОУ, Адреса, Банківські реквізити
```

**2. Додай клієнтів (синхронізуються з CRM)**
```
Books → Контакти → Імпорт з CRM (якщо вже є в CRM)
```

**3. Створи перший інвойс**
```
Books → Продажі → Інвойси → Новий інвойс
Вкажи: Клієнт, Послуги/роботи, Сума, Дата оплати
```

**4. Налаштуй категорії витрат**
```
Матеріали / Субпідряд / Зарплата / Техніка / Адмін витрати
```

---

## Крок 4 — Zoho Projects (проекти)

### Що це дає
- Кожен будівельний об'єкт — окремий проект
- Задачі, терміни, виконавці
- Гантт-діаграма (хто що робить і коли)
- Прив'язка до клієнта в CRM

### Перше налаштування (20 хвилин)

**1. Створи шаблон будівельного проекту**
```
Projects → Шаблони → Новий шаблон "Будівельний об'єкт"
Задачі: Проектування → Дозволи → Фундамент → Каркас → 
        Покрівля → Інженерія → Оздоблення → Здача
```

**2. Запусти перший проект з шаблону**
```
Projects → Новий проект → З шаблону
Вкажи: Назва об'єкту, Клієнт, Дата початку, Бюджет
```

---

## Крок 5 — Zoho Sign (договори)

### Що це дає
- Підписання договорів онлайн без зустрічі
- Клієнт отримує email → підписує з телефону
- Юридична сила (за законом України про е-документи)

### Як використати
```
Sign → Новий документ → Завантаж договір (PDF/Word)
→ Вкажи поля для підпису → Відправ клієнту на email
```

## Інтеграція між модулями

```
CRM Угода виграна
        ↓
Books автоматично створює інвойс
        ↓
Projects автоматично створює проект
        ↓
Sign відправляє договір на підпис
```

Налаштовується через **Zoho Flow** (аналог Make/Zapier, але всередині Zoho).
